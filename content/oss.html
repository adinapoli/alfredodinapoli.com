Open Source

Independent projects, Haskell contributions, and tooling. See GitHub for the full list.

// independent projects

  • Rusholme — a toy Haskell compiler in Zig, in development. Frontend, Core, GRIN, LLVM/WASM/C backends. zig llvm grin
  • mandrill — Mandrill transactional email API in Haskell. haskell
  • rncryptor-hs — RNCryptor encryption format, ported to Haskell. security
  • snaplet-purescript — a Snap snaplet for compiling PureScript as part of your build. haskell
  • threads-supervisor — Erlang-style supervisors for Haskell threads. haskell

  • GHC diagnostic infrastructure — contributed to GHC's new structured diagnostic system at Well-Typed. haskell ghc
  • Snap framework — implemented the Snap.Snaplet.Test module. haskell
  • reactive-banana — minor contributions. haskell
