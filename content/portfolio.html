Laetus Framework

Laetus is a parallel C++ framework that aims to alleviate the burder of writing plain OpenCL, providing an higher level of abstraction and several improvements over the standard computational model of OpenCL. Laetus is my senior thesis project.

Clj3D Library

Clj3D is a Clojure graphic library for manipulating 3D and 2D objects. It aims to be the Clojure standard in 3D and 2D rendering.



Newsface
