> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

Replace entire file with:

```haskell
{-# LANGUAGE OverloadedStrings #-}

module Main where

import qualified Data.Text.IO    as T
import           System.Environment (getArgs)
import           System.Exit        (exitWith, ExitCode(..))
import           System.Process     (system)

usage :: IO ()
usage = T.putStrLn "usage: site-ctl [build|watch|clean|deploy]"

run :: String -> IO ()
run cmd = system cmd >>= exitWith

main :: IO ()
main = do
  args <- getArgs
  case args of
    ["build"]  -> run "cabal run site -- build"
    ["watch"]  -> run "cabal run site -- watch"
    ["clean"]  -> run "cabal run site -- clean"
    -- "deploy" intentionally omitted — owner wires their own deploy
    -- step (gh-pages push, rsync, etc.) outside this binary.
    _          -> usage >> exitWith (ExitFailure 64)
```

(Deploy is intentionally out of scope here — the owner runs their own deploy step.)
